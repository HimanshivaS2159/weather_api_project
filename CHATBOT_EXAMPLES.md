## 🎨 Formatting Examples

The chatbot now supports:

### ✅ Tables (Markdown format)
```
| City | Temp | Humidity |
|------|------|----------|
| Mumbai | 32°C | 78% |
```

### ✅ Bold Text
Use `**text**` → **text**

### ✅ Code/Technical Terms
Use `` `term` `` → `term`

### ✅ Lists
Use `- item` for bullet points:
- Item 1
- Item 2
- Item 3

## 🚀 Tips for Best Results

1. **Ask for comparisons** - Chatbot will create tables automatically
2. **Use "table format" or "comparison"** in your question
3. **Be specific** - "Compare 3 cities" works better than "tell me about cities"
4. **Ask follow-up questions** - Build on previous answers

## 🎯 Pro Tips

- Tables work best for **2-5 columns**
- Chatbot automatically formats **comparison** requests as tables
- Use **bold** and `code` for emphasis
- Ask for **"show in table"** to force table format
